# Planos y Presupuesto - Laboratorio Microbiología BSL-2

## VeriTest.LAB

Repositorio para almacenar los planos arquitectónicos, de instalaciones y presupuesto detallado del laboratorio.

### Estructura

```
planos/
├── 01-arquitectura/      # Planos arquitectónicos
├── 02-instalaciones/     # Eléctrica, plumbing, HVAC, datos
├── 03-equipamiento/     # Distribucion de equipos
├── 04-presupuesto/      # Presupuesto detallado por capítulos
└── 05-documentacion/    # Memorias, cálculos, especificaciones
```

### Objetivo

Desarrollar la documentación técnica completa para:
1. Licencia de apertura
2. Construcción/obra civil
3. Instalación de equipamiento
4. Certificación BSL-2
